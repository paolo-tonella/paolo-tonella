Tools

  • ROBULA: Automated creation of robust locators for web testing. This tool generates xpath locators that target web page elements used in test case actions or assertions. The generated locators are optmized to be resilient to the evolution of the web application under test. DOI
  • EvoSuite-MOSA: Automated test case generation for unit testing of Java classes. MOSA extends EvoSuite with a novel many objective algorithm, which achieves high coverage very efficiently. DOI
  • ReAjax: Reverse engineering and testing of Ajax web applications. This tool extracts a finite state model of the Ajax web application under test by dynamic anaysis, and uses the model for test case derivation. DOI
  • eToc: Tool for evolutionary generation of object-oriented test cases. Presented at ISSTA 2004, this tool initiated a new stream of research on automated unit testing that produced, among others, EvoSuite, a mature and advanced Java test case generator. DOI
  • RuleChecker: Automated verification of the code compliance to coding conventions and detection of programming anti-patterns. The tool has been routinely used at CERN, Geneva, to check the quality of the C++ software for data analysis and physics simulation. LINK
  • ReWeb and TestWeb: Based on the algorithms described in the ICSE MIP award winning paper "Analysis and Testing of Web Applications", these two tools support the creation of a web application model and the derivation of test cases from such model. DOI
